Getting Started

Each of the challenges in this section will be contained in its own folder in this project. The folder will contain an index.html file like this one, and an app.js file which is where you will write your JavaScript code to complete the challenge.

The HTML page will always have a pre element with an id of output-content. You can see this element below, and you can use it or the console.log() method to see the output your code is producing. You should check out the app.js file now to see how the this works.
